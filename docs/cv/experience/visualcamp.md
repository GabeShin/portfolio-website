# VisualCamp

## What VisualCamp does

Provides a mobile gaze-tracking SDK used by developers worldwide.

## My time there

Joined as ML Researcher when the team was small, grew with the product into Tech Lead and then Backend Engineer roles. Owned the SeeSo SDK roadmap during the platform expansion phase, contributed to the C++ core, and built out the cloud infrastructure that the product runs on today.

## Roles in detail

**Backend Engineer** (2022-09 – 2024-05)

- Built and operated scalable backend services on Kubernetes.
- Implemented monitoring and observability with Fluentd and Grafana.
- Designed event-driven backend workflows for asynchronous operations while preserving data integrity.
- Applied TDD across the team to improve maintainability.

**Tech Lead** (2020-03 – 2022-09) — formal title with direct reports

- Led the SeeSo SDK platform roadmap; expanded support to six platforms and supported 5M+ monthly authentications.
- Refactored a legacy monolithic authentication server into a microservices architecture, reducing server cost by approximately 70%.
- Designed and deployed global SDK authentication backed by CDN.
- Introduced infrastructure-as-code with AWS CDK as the team standard.
- Contributed to the C++ SDK core and TensorFlow integration for cross-platform inference.

**Machine Learning Researcher** (2018-09 – 2020-03)

- Researched and reproduced state-of-the-art gaze-tracking models; translated findings into product improvements.
- Partnered with the engineering team on crowdsourced data collection.
- Improved model accuracy by 25%, contributing to state-of-the-art performance in mobile gaze tracking.
